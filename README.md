## Getting Started


### Installation

1. Clone the repository:
```bash
git clone https://github.com/Sanskar2709/dynamic-report-dashboard.git
cd dynamic-report-dashboard
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Run tests:
```bash
npm run test:e2e
```
